Customize and optimize comments in WordPress

== Description ==

Fine tune comments in WordPress.

You can find the settings page under `Comments > Settings` in the dashboard.

**Features**

* remove comments from post type: posts, pages, attachments and other public custom post types
* [more coming soon]

I am open to your suggestions for more features here or in the [github repository](https://github.com/webgilde/comments)

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit the settings page under `Comments > Settings` to customize the form
